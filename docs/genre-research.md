# ジャンルリサーチ: AI×副業 / AI×ツール解説(2026年7月時点)

計画書(`youtube-monetization-plan.md`)の前提となる市場調査。結論から言うと、**ジャンル自体は伸びているが「AI丸投げ量産」は規制強化で淘汰が始まっており、"人間の検証・体験"を入れた動画だけが生き残る**局面。当チャンネルの「実証型」戦略はこの流れに合致している。

---

## 1. 市場の現状(2026年)

### 追い風

- 顔出しなし×AI活用チャンネルはもはやニッチではなく、誰でも高品質なコンテンツを作れる環境が整った
- 解説・教育系は安定した需要と高い広告単価(CPMはジャンルにより平均100〜500円)がある
- VOICEVOX(ずんだもん等)を使った解説チャンネルで「初投稿から約50日で収益化」「3ヶ月で登録者2,000人」「33日で登録者2万人」などの事例が報告されている
- ChatGPTの使い方・AIツール紹介チャンネルが「開設2ヶ月で登録者2.8万人」に達した事例もあり、AIツール解説の検索需要は依然強い

### 向かい風(重要)

- **2026年始、YouTubeショートでAI量産動画の収益化停止・アカウント削除が続出**。削除されたのは「他人の投稿をまとめただけ」「作り手の顔が見えない機械的な動画」
- 原因は「コモディティ化」— 誰でもAIで同じような動画を作れるため、差別化のない動画は視聴者にもアルゴリズムにも選ばれなくなった
- 一方で「人間の視点・構成・体験・解釈が入った動画」は2026年現在も収益化できているチャンネルが多数

### 結論

「AIに丸投げ」は死亡、「人間が主導してAIを使う」はまだ大きなチャンス。当チャンネルの方針:

1. **実証型**(実際にやって数字を公開)を全動画の軸にする — コモディティ化の対極
2. ショート依存にしない(規制リスクが集中している)— 長尺をチャンネルの本体に
3. AI音声(VOICEVOX)は問題なし。ただし台本・構成・検証に必ず自分の体験を入れる

## 2. 競合チャンネルの傾向

| タイプ | 例 | 学べること |
|---|---|---|
| ビジネス系AI解説(顔出し・専門家型) | 週1〜2本ペースでAI基礎からツール活用まで網羅する法人研修系チャンネル | 信頼性で勝負。個人が真正面から戦う場所ではない |
| 初心者向けツール解説 | いけともch(ChatGPT・リモートワークツールをIT苦手層向けに解説) | 「対象者を絞って噛み砕く」だけで差別化になる |
| 急成長型ツール紹介 | ChatGPT使い方・AIツール紹介で開設2ヶ月2.8万人 | 「新ツール速報×使い方」は今も伸びる型 |
| VOICEVOX解説(非属人) | 歴史系ずんだもん解説で3ヶ月収益化・売買市場でも取引される | 非属人でも収益化可能。テンポと投稿頻度(週2〜3本)が生命線 |

**空いているポジション**: 「AI副業を実際にやって、収支をありのまま公開する初心者目線の検証チャンネル」。成功だけ語るチャンネルは飽和しているが、失敗込みの検証記録は少なく信頼を取りやすい。

---

## 3. 数字の目安(事例ベース)

- 収益化ライン: 登録1,000人+再生4,000時間(変更可能性あり、開始時に公式確認)
- CPM: 解説・教育・ビジネス系で100〜500円/1,000再生(エンタメ系より高い)
- 投稿頻度: 最低週1、理想は週2〜3本(収益化済みチャンネルの共通項)
- 収益化までの期間: 型がハマれば2〜3ヶ月の事例あり。計画書の「6ヶ月で収益化ライン」は保守的で妥当

---

## 4. リスクと対策

| リスク | 対策 |
|---|---|
| AI量産系への規制強化(再利用コンテンツ判定) | 全動画に自分の検証・数字・解釈を入れる。台本の丸ごとAI生成でも「内容」が自分の体験なら問題ない |
| ショートの収益化停止リスク | ショートは集客用と割り切り、収益の柱は長尺+アフィリエイトに置く |
| ジャンルのコモディティ化 | 「実証型」+「初心者特化」の二重の絞り込みで型を作る |
| VOICEVOX規約 | キャラごとの利用規約とクレジット表記を確認(ずんだもんはYouTube収益化OK) |
