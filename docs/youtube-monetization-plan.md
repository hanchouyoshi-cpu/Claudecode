# YouTube「AI×副業 / AI×ツール解説」ゼロから収益化プラン

- **目標**: 月5万円の収益(広告+アフィリエイト)
- **テーマ**: AI×副業・お金稼ぎ(メイン)+ AI×ツール解説(サブ)の2本柱
- **スタイル**: 顔出しなし(AI音声+スライド解説)
- **稼働**: 週15時間以上
- **環境**: Mac + Claude(+無料ツール中心)

> 関連資料: 市場調査は [`genre-research.md`](genre-research.md)、制作の自動化スクリプトは [`../automation/`](../automation/README.md) を参照。

## 1. チャンネルコンセプト設計

### 1-1. 2本柱の役割分担

| 柱 | 役割 | 特徴 |
|---|---|---|
| **AI×副業・お金稼ぎ** | 集客の柱(バズ・おすすめ流入) | 需要が大きく伸びやすい。競合も多いので「実際にやってみた+数字公開」で差別化 |
| **AI×ツール解説** | 資産の柱(検索・関連動画流入) | 「ChatGPT 使い方」「Claude 比較」等の検索需要で長期間再生され続ける。アフィリエイトと相性が良い |

**運用イメージ**: 副業系でチャンネルに人を集め、ツール解説系で「登録して継続視聴する理由」と「アフィリエイト収益」を作る。動画の最後で相互に誘導する。

### 1-2. ターゲット視聴者

- **メイン**: 20〜40代の会社員。「副業を始めたいがスキルも時間もない」「AIを使えば楽にできるのでは」と考えている層
- **視聴者の欲求**: ①具体的な手順が知りたい ②本当に稼げるのか実例が見たい ③難しい専門用語なしで理解したい

### 1-3. ポジショニング(差別化)

競合が多いジャンルなので、以下のどれかを必ず動画に入れる:

1. **実証型**: 「実際に1ヶ月やって○○円稼げた/稼げなかった」と数字を出す(失敗も出すと信頼される)
2. **手順完全公開型**: プロンプトや設定を画面収録でそのまま見せる(「概要欄にプロンプト置いてます」で登録・コメント誘導)
3. **初心者特化型**: 前提知識ゼロで再現できるレベルまで噛み砕く

### 1-4. チャンネル名・肩書きの決め方

- 「何のチャンネルか3秒でわかる」を最優先。例:「○○(名前)|AI副業ラボ」「AIで稼ぐ研究所」
- 検索されたいキーワード(AI・副業)をチャンネル名か概要文の冒頭に入れる
- 肩書きは盛らない。「会社員をしながらAI副業を検証中」のような等身大設定のほうが顔出しなしでも信頼を得やすい

## 2. 制作ワークフロー(Mac + Claude で完結・顔出しなし)

### 2-1. 使用ツール一覧

| 工程 | ツール | 費用 | 備考 |
|---|---|---|---|
| 企画・リサーチ | Claude + YouTube検索・急上昇 | 無料〜 | 競合動画のタイトル・構成分析もClaudeに手伝わせる |
| 台本 | Claude | 無料〜 | 後述のテンプレプロンプトを使用 |
| AI音声 | **VOICEVOX**(Mac対応・無料・商用利用可) | 無料 | クレジット表記(例:「VOICEVOX:ずんだもん」)が必要。キャラごとの利用規約を確認 |
| 音声(代替) | 音読さん / CoeFont | 無料枠あり | より自然な声が欲しい場合。無料枠の商用条件を確認 |
| スライド | Keynote(Mac標準)or Canva | 無料 | 1本目はKeynoteで十分。テンプレを1つ作って使い回す |
| 画面収録 | macOS標準(Cmd+Shift+5)or QuickTime | 無料 | ツール解説の操作画面撮影に使用 |
| 編集 | **CapCut**(簡単)or DaVinci Resolve(高機能) | 無料 | 最初はCapCut推奨。字幕自動生成が使える |
| サムネイル | Canva | 無料 | AI画像生成もCanva内で可能 |
| BGM・効果音 | YouTubeオーディオライブラリ / DOVA-SYNDROME | 無料 | 著作権フリー素材のみ使う |

### 2-2. 台本作成プロンプト(Claude用テンプレ)

```text
あなたはYouTube台本作家です。以下の条件で解説動画の台本を書いてください。

# 動画テーマ
(例: ChatGPTを使ってブログ記事を量産し、月1万円稼ぐ手順)

# 条件
- 想定視聴者: AI初心者の会社員(専門用語は都度ひとこと解説)
- 長さ: 10分(読み上げで約3,000〜3,500字)
- 構成:
  1. フック(15秒以内): 結論とベネフィットを先に言う
  2. この動画で得られること(3点)
  3. 本編(手順を番号付きで、画面操作の指示も書く)
  4. 注意点・失敗しやすいポイント
  5. まとめ+次に見るべき動画への誘導+チャンネル登録の一言
- 文体: です・ます調、1文は短く、話し言葉
- 各セクションに [スライド: 〇〇] の形でスライド指示を入れる
```

### 2-3. 1本あたりの想定工数(長尺10分動画)

| 工程 | 初期 | テンプレ化後 |
|---|---|---|
| 企画・リサーチ | 1.5h | 0.5h |
| 台本(Claude+手直し) | 1.5h | 1h |
| スライド作成 | 2h | 1h |
| 音声生成・画面収録 | 1h | 0.5h |
| 編集・字幕 | 3h | 2h |
| サムネ・タイトル・概要欄 | 1h | 0.5h |
| **合計** | **約10h** | **約5.5h** |

**短縮のコツ**: ①スライドのマスターテンプレを固定 ②動画のオープニング/エンディングを共通素材化 ③サムネもレイアウト固定で文字だけ差し替え ④Claudeへの指示文を保存して使い回す。

さらに、リサーチ・台本生成・音声合成はスクリプトで自動化済み(`automation/` 参照)。自動化後は1本あたり約4時間まで短縮できる想定。

## 3. 投稿戦略(週15時間の配分)

### 3-1. 投稿ペース

- **長尺(8〜15分)**: 週2本 — チャンネルの本体。収益化条件の「再生4,000時間」はほぼ長尺で稼ぐ
- **ショート(60秒以内)**: 週3〜5本 — 長尺の要点切り出し+テキスト系小ネタ。登録者獲得と露出用

### 3-2. 週間スケジュール例(計15h)

| 曜日 | 作業 | 時間 |
|---|---|---|
| 月 | 企画・リサーチ(2本分)+台本1本目 | 2.5h |
| 火 | 台本2本目+スライド1本目 | 2.5h |
| 水 | スライド2本目+音声・収録 | 2.5h |
| 木 | 編集1本目 | 2.5h |
| 金 | 編集2本目+サムネ2枚 | 2.5h |
| 土 | ショート3〜5本切り出し+予約投稿設定 | 1.5h |
| 日 | 分析(アナリティクス確認)・翌週ネタ決め | 1h |

- 投稿は**予約投稿**で曜日・時間を固定(例: 火・金の18時)。視聴習慣がつきやすく、アルゴリズム的にも安定する

## 4. 収益化ロードマップ(0→3→6→12ヶ月)

### 収益化の条件(YouTubeパートナープログラム)

- 広告収益: **登録者1,000人 + 直近12ヶ月の総再生4,000時間**(または直近90日のショート再生1,000万回)
- ※条件は変更されることがあるため、開始時に公式ヘルプで最新条件を確認すること

### フェーズ1(0〜3ヶ月): 型を作る — 目標: 動画30本・登録300人

- とにかく30本投稿し、データを貯める(最初の10本は伸びなくて正常)
- 毎週アナリティクスで **CTR(サムネクリック率)** と **視聴維持率** を確認
  - CTR 4%未満 → サムネ・タイトルを改善
  - 冒頭30秒の離脱が多い → フックの作り方を改善
- 伸びた動画の「型」(テーマ×タイトル×構成)を特定し、類似企画を重ねる

### フェーズ2(3〜6ヶ月): 収益化ライン到達 — 目標: 登録1,000人・4,000時間

- フェーズ1で見つけた「勝ちパターン」に投稿の7割を寄せる
- 収益化**前**からアフィリエイトは可能なので、ツール解説動画の概要欄に導線を設置(先に収益発生させる)
- 過去動画のテコ入れ: 伸びかけている動画のサムネ差し替え、終了画面の相互リンク整備

### フェーズ3(6〜12ヶ月): 月5万円達成

**収益ミックスの試算(目安)**:

| 収益源 | 前提 | 月収目安 |
|---|---|---|
| 広告収益 | 月10万再生 × 単価0.2〜0.4円(ビジネス系は単価高め) | 2〜4万円 |
| アフィリエイト | AIツール・スクール案件(A8.net、もしもアフィリエイト等)。成約単価1,000〜10,000円 × 月5〜20件 | 1〜5万円 |
| その他導線 | note/Kindle(プロンプト集・ノウハウ)、メンバーシップ | 0〜数万円 |

→ **広告2〜4万+アフィリエイト1〜5万で月5万円ラインを狙う**。広告単価が低くても届く設計にしておくのがポイント。

### KPIまとめ

| 時期 | 投稿数(累計) | 登録者 | 重点KPI |
|---|---|---|---|
| 3ヶ月 | 30本 | 300人 | CTR 4%以上 / 視聴維持率40%以上 |
| 6ヶ月 | 55本 | 1,000人 | 収益化申請 / アフィ成約 月5件 |
| 12ヶ月 | 100本 | 3,000〜5,000人 | 月収5万円(広告+アフィ) |

## 6. 継続のための仕組み

### 6-1. 週次振り返りテンプレ(日曜1時間)

```text
1. 今週の投稿: 本数 / 各動画の再生数・CTR・維持率
2. 一番伸びた動画とその理由(仮説)
3. 一番伸びなかった動画とその理由(仮説)
4. 来週試すこと(1つだけ変える: サムネ / タイトル / 冒頭 / テーマ)
5. 収益: 広告 ___円 / アフィ ___円
```

### 6-2. ネタ切れ防止のリサーチ法

- 競合チャンネルの「人気動画」タブを見て、1年前の人気テーマを最新情報で作り直す
- X(Twitter)やYouTubeの検索サジェスト(「AI 副業 」の後に続く候補)をネタ帳にストック
- 視聴者コメントの質問はすべて企画候補としてメモ
- Claudeに「このチャンネルの視聴者が次に知りたいことを20個」とブレストさせる

### 6-3. やってはいけないこと(重要)

- **誇大表現**: 「誰でも確実に月100万」等は視聴者の信頼を失い、広告主にも嫌われる。実数字ベースで話す
- **AI音声・素材の規約違反**: VOICEVOXはキャラクターごとに利用規約が異なる。クレジット表記と商用条件を必ず確認
- **他人の動画・記事の丸写し**: 再利用コンテンツと判定されると収益化審査に落ちる。必ず自分の検証・言葉を入れる
- **YouTubeポリシー違反**: 金融・稼ぐ系はガイドラインが厳しめ。「投資助言」に踏み込まない、リンク先の案件の信頼性を確認する
- **アフィリエイトのステマ**: PR案件・アフィリエイトリンクはその旨を明記(ステマ規制対応)
- **完璧主義**: 最初の10本はクオリティより完成・公開を優先。改善はデータが出てから
